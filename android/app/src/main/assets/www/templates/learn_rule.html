得分规则

最终得分，为以下各项加权得分之和，满分100分

视频：

结课前看完全部视频，才可以拿到视频成绩！

得分规则：(完整观看视频数/总视频数) x 权重{{proportion.videoPercent}}%

作业：

老师作业以老师评分为准；互评作业在互评结束后公布成绩。

得分规则：(作业得分总和/总作业数) x 权重{{proportion.assignmentPercent}}%

讨论：

截止前参与才有成绩，默认满分，最终以老师参与的评分为准。

得分规则：(讨论得分总和/总讨论数) x 权重{{proportion.topicPercent}}%

测验：

如果是能多次提交的测验，以最高分为准。

得分规则：(测验得分总和/总测验数) x 权重{{proportion.quizPercent}}%

考试：

考试成绩在成绩公布后可查看，若参加重考，以重考成绩为准。

得分规则：考试成绩 x 权重{{proportion.examPercent}}%

线下成绩：

线下成绩考核标准以老师通知为准，由老师录入系统后显示。

得分规则：线下成绩 x 权重{{proportion.offlinePercent}}%
